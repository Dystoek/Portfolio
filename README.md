## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... more variables */
}
```

### Content
- Edit `index.html` to update text content
- Modify sections as needed
- Add or remove projects/skills

### Typing Animation
Edit the `typingTexts` array in `script.js`:
```javascript
const typingTexts = [
    'AI/ML Researcher',
    'Data Scientist',
    // Add your own text here
];
```
